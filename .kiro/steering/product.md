# Product Overview

YouTube Downloader Pro is a desktop application for downloading YouTube videos, playlists, and audio content. Built with a modern GUI using PySide6 and PyQt-Fluent-Widgets, it provides an intuitive interface for managing downloads with features like batch processing, quality selection, format conversion, and download history tracking.

## Core Features

- Single video and batch download support
- Playlist download with concurrent processing
- Multiple format support (MP4, WebM, MKV, MP3, M4A, WAV, FLAC)
- Quality selection from 144p to best available
- Audio-only extraction
- Subtitle and thumbnail downloads
- Download history tracking
- Resume and skip existing file capabilities
- Dark/Light theme support with system theme detection

## Target Users

Users who need to download YouTube content for offline viewing, archival, or audio extraction purposes while respecting copyright and YouTube's Terms of Service.
