# FluentIcon Reference

This document lists all available FluentIcon constants that can be used in the YouTube Downloader application.

## Usage

Import FluentIcon in your Python files:
```python
from qfluentwidgets import FluentIcon as FIF
```

Then use icons like: `FIF.DOWNLOAD`, `FIF.LIBRARY`, `FIF.HISTORY`, etc.
